# Alex's Minecraft Shaders

Welcome to **Alex's Minecraft Shaders**! This repository will be a growing collection of really good, highly optimized Minecraft shaders designed to bring beautiful visuals to the game without sacrificing performance. 

## Current Shaders

### Cinematic Mac (Optimised)
A beautiful, cinematic shader built from the ground up for maximum performance on Apple Silicon (M1/M2/M3) Macs.

**Features:**
- **Target Performance:** Designed to hold 50-60 FPS on an M2 MacBook Pro with 8GB RAM.
- **Stylized Atmospheric Scattering:** Beautiful skies simulating Rayleigh and Mie scattering without the extreme performance cost of full volumetric raymarching.
- **2.5D Procedural Clouds:** Stunning, dynamic clouds driven by Fractional Brownian Motion (FBM) noise.
- **ACES Tonemapping:** Film-like color grading for vibrant, realistic colors and a subtle vignette.
- **High Compatibility:** Built to run flawlessly on both **OptiFine** and **Iris Shaders** across modern Minecraft versions (1.16.5 - 1.21+).

## Installation
1. Download or clone this repository.
2. Compress the specific shader folder you want to use (e.g., `Cinematic Mac (optimised)`) into a `.zip` file, or simply copy the folder itself.
3. Move the `.zip` or folder into your Minecraft `shaderpacks` directory:
   - **Mac:** `~/Library/Application Support/minecraft/shaderpacks/`
   - **Windows:** `%appdata%\.minecraft\shaderpacks\`
4. Launch Minecraft using Iris or OptiFine, navigate to `Video Settings > Shaders`, and select the pack!
